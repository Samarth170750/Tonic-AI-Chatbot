# Tonic AI Chatbot

Tonic is an AI-powered chatbot built using Python, Flask, and Google Gemini API. It provides intelligent and structured responses in JSON format while maintaining conversation memory to deliver context-aware and consistent interactions.

The chatbot can assist users with programming, concept explanations, problem-solving, and general conversations. By leveraging chat history, Tonic remembers recent interactions and generates more relevant responses throughout the conversation.
